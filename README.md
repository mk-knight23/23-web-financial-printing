# MK ChequeCraft

**Privacy-First Cheque & Financial Printing**

Generate and print bank cheques privately in your browser. No data leaves your device.

🔗 **Live**: [https://23-web-financial-printing.vercel.app](https://23-web-financial-printing.vercel.app)

## Tech Stack

Angular 21, TypeScript, Tailwind CSS 4

## Getting Started

```bash
npm install
npm run dev
```
