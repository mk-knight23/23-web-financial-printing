# Product Specification - MK ChequeCraft

- **Core Problem:** Users need a simple, fast, free, registration-free utility to solve: "Privacy-first cheque and financial printing utility".
- **Target Audience:** Everyday internet users, developers, and professionals.
- **Key Workflows:**
  1. Instant utility access on page load (simple first).
  2. Advanced configuration options via drawers/tooltips (progressive disclosure).
  3. Safe localStorage or IndexedDB persistence.
- **Features Included:** Position bank-cheque fields, print calibration, convert numeric amount to words, batch csv imports, local storage drafts.
